## 프로젝트 실행 방법

```bash
npm install

echo "VITE_API_BASE_URL=https://fe-hiring-rest-api.vercel.app" > .env

npm run dev
```

##  사용한 기술 스택

### Core
- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구

### Routing & State
- **React Router v6** - 클라이언트 사이드 라우팅

### Styling
- **Tailwind CSS** - 유틸리티 기반 스타일링

### Form & Validation
- **React Hook Form** - 폼 상태 관리
- **Zod** - 스키마 기반 유효성 검사

### Data Visualization
- **Recharts** - 선언적 차트 라이브러리

### HTTP Client
- **Axios** - API 통신

### Code Quality
- **ESLint** - 코드 품질 검사
- **Prettier** - 코드 포매팅

## 주요 구현 기능 요약
- 모든 페이지 반응형
- 보호된 라우트 리다이렉트

### 홈 페이지 ("/")
#### 네비게이션
- 게시글 목록 페이지
- 데이터 시각화 페이지
#### 기능
- 로그인 / 로그아웃

### 게시글 목록 페이지 ("/posts")
#### 네비게이션
- 홈 페이지
- 게시글 상세 페이지
- 게시글 작성 페이지
#### 기능
- 게시글 목록 조회
- 테이블 컬럼 너비 조절
- 컬럼 숨김 / 보임
- 무한 스크롤
- 제목 및 본문 내용 검색 
- title 또는 createdAt 기준 오름/내림차순 정렬
- 카테고리(category)별 필터링

### 게시글 상세 페이지 ("/posts/${id}")
#### 네비게이션
- 홈 페이지
- 게시글 목록 페이지
- 게시글 수정 페이지
#### 기능
- 게시글 상세 조회
- 게시글 삭제

### 게시글 수정 페이지 ("/posts/${id}/edit")
#### 네비게이션
- 홈 페이지
- 게시글 목록 페이지
- 게시글 상세 페이지
#### 기능
- 게시글 수정
- 금칙어 및 폼 유효성 관리

### 게시글 작성 페이지 ("/posts/new")
#### 네비게이션
- 홈 페이지
- 게시글 목록 페이지 
#### 기능
- 게시글 작성
- 금칙어 및 폼 유효성 관리

### 데이터 시각화 페이지 ("/charts")
#### 네비게이션
- 홈 페이지
#### 기능
- 6가지 실시간 데이터 기반 인터랙티브 차트 제공
- 범례에서 색상 실시간 변경 
- 범례에서 항목 가시성 토글
- 숨겨진 항목은 차트에서 즉시 사라지고, 색상은 유지됨
#### 차트 목록
| 차트 이름                | 타입                              | 주요 특징                                       |
|--------------------------|-----------------------------------|---------------------------------------------|
| 커피 브랜드 인기도       | 바 차트 + 도넛 차트               | 브랜드별 인기도 점유율 시각화                            |
| 인기 스낵 브랜드         | 바 차트 + 도넛 차트               | 스낵 브랜드별 시장 점유율                              |
| 주간 기분 트렌드 (%)     | 스택형 바 차트 + 스택형 면적 차트 | happy / tired / stressed 비율을 100% 스택으로 표시   |
| 주간 운동 트렌드 (%)     | 스택형 바 차트 + 스택형 면적 차트 | running / cycling / stretching 비율을 주차별로 비교  |
| 커피 소비와 생산성       | 듀얼 Y축 멀티 라인 차트           | 왼쪽 축: 버그 수, 오른쪽 축: 생산성, 호버 시 팀 데이터만 툴팁      |
| 스낵 섭취 영향           | 듀얼 Y축 멀티 라인 차트           | 왼쪽 축: 회의 불참 건수, 오른쪽 축: 사기 지수, 호버 시 해당 부서 데이터만 툴팁 |
